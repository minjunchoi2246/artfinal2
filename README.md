# Korean Highway Rest Stop Food Dashboard

A national Streamlit dashboard for Korean highway rest stop food information.

## Run locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Main updates in v5

- National dashboard structure restored.
- Dashboard View has only two options: `교통량 많은 휴게소` and `전부 보기`.
- Rest stop names and menu names are written in Korean.
- Phone number fields were removed.
- Map popup text was enlarged and simplified; full details are shown in the Streamlit detail panel for better readability.
- Each rest stop includes menu descriptions, food area types, prices, hours, and rating input.
